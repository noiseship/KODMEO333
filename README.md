**Автор:** Вавилин Матвей

---

## 📝 Описание

Expense Tracker — это десктопное GUI-приложение для отслеживания личных расходов.
Позволяет добавлять расходы по категориям, фильтровать по датам,
просматривать статистику и автоматически сохранять данные в JSON-файл.

**Технологии:** Python 3, Tkinter, JSON, unittest

## 💾 Как работает сохранение данных (JSON)

Все расходы автоматически сохраняются в файл `expenses.json` в папке проекта.

### Формат файла `expenses.json`:

```json
[
  {
    "id": 1,
    "amount": 500.0,
    "category": "Food",
    "date": "2026-04-30"
  },
  {
    "id": 2,
    "amount": 150.0,
    "category": "Transport",
    "date": "2026-05-01"
  }
]
```

### Поля записи:
| Поле       | Тип    | Описание                        |
|------------|--------|---------------------------------|
| `id`       | int    | Уникальный идентификатор записи |
| `amount`   | float  | Сумма расхода в рублях          |
| `category` | string | Категория расхода               |
| `date`     | string | Дата в формате `YYYY-MM-DD`     |

### Чтение и запись в коде:

```python
# Сохранение (запись в файл)
with open("expenses.json", "w", encoding="utf-8") as f:
    json.dump(self.expenses, f, ensure_ascii=False, indent=2)

# Загрузка (чтение из файла)
with open("expenses.json", "r", encoding="utf-8") as f:
    self.expenses = json.load(f)
```

> ⚠️ Файл `expenses.json` добавлен в `.gitignore` и не попадает в репозиторий —
> у каждого пользователя будут свои локальные данные.

## 📖 Примеры использования

### Пример 1: Добавление расхода
1. Введите сумму в поле **Amount (RUB)**, например: `350`
2. Выберите категорию в **Category**, например: `Food`
3. Укажите дату в поле **Date**, например: `2026-05-01`
4. Нажмите кнопку **Add Expense**
5. Расход появится в таблице, данные сохранятся автоматически

### Пример 2: Фильтрация по категории
1. В разделе **Filter** выберите категорию, например: `Transport`
2. Нажмите **Apply Filter**
3. В таблице останутся только расходы на транспорт

### Пример 3: Фильтрация по периоду
1. В поле **Date from** введите: `2026-04-01`
2. В поле **Date to** введите: `2026-04-30`
3. Нажмите **Apply Filter** — отобразятся расходы только за апрель

### Пример 4: Удаление расхода
1. Кликните на строку в таблице, чтобы выбрать её
2. Нажмите кнопку **Delete Selected**
3. Подтвердите удаление в диалоговом окне

### Пример 5: Проверка валидации (ввод ошибочных данных)
| Ввод               | Результат                                      |
|--------------------|------------------------------------------------|
| Сумма: `-100`      | Ошибка: «Amount must be positive»              |
| Сумма: `abc`       | Ошибка: «Amount must be a number»              |
| Дата: `30-04-2026` | Ошибка: «Invalid date format. Use YYYY-MM-DD»  |
| Сумма: пусто       | Ошибка: «Amount must be a number»              |
